Cách Chạy:
1. Bỏ video gốc vào "input"
2. Chạy codeo "AddVien.bat" hoặc "AddVien+Latngang.bat"
3. Copy "Creatsub.py" 
4. Vào đường dẫn thư mục gõ "cmd" rồi Enter, sau đó paste "Creatsub.py" và Enter để chạy.

Môi Trường:
pip install python-dotenv
pip install moviepy==1.0.3
pip install assemblyai
